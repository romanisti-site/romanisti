# Release Notes — Build 1.0.6

Build 1.0.6 shifts Project Faber decisively toward the reader.

Observations can now be explored through a living topic taxonomy. Each article
can continue into related work rather than ending as an isolated page.

The canonical Codex also receives its first dedicated reading experience:
contents, book navigation, restrained typography, and a complete five-plate
closing sequence built around the preserved Edition 1.0A manuscript.
