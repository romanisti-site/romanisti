# Release Notes — Repository 1.0.1

This is the first complete Project Faber repository release.

It converts Romanisti from constitutional inheritance into a deployable publishing application while preserving the boundary between institutional source material and implementation.

The release provides a functioning editorial pipeline from typed Markdown source to index pages, canonical article routes, metadata, feeds, and static deployment.

The `/library/` route is intentionally present before canonical manuscript import. The route is real; the locked texts are not recreated from memory.
