# Release Notes — Build 1.0.4

Build 1.0.4 turns the Project Faber foundation into a more complete editorial
experience.

The homepage now behaves as an institutional front door rather than a technical
demonstration. It introduces the publication, presents the featured
observation, leads readers into the current record, and connects the public
site to the canonical Library.

This release also establishes the normal update cadence: a guarded incremental
package applied to a clean local `faber` branch, followed by review in GitHub
Desktop.
